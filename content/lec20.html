Lecture 20: CSS Grid Layout

Summary

المحاضرة دي كانت عن واحدة من أقوى وأحدث أدوات تنسيق الصفحات في CSS وهي Grid Layout. اتعلمنا إزاي ننظم العناصر داخل الصفحة في شبكة مكونة من صفوف وأعمدة بطريقة مرنة وسهلة، بدل ما نستخدم float أو flexbox في التنسيقات المعقدة. شفنا كمان إزاي نتحكم في حجم الأعمدة والصفوف، والمسافات بينهم، وإزاي نخلي عناصر معينة تمتد على أكتر من عمود أو صف.

Key Points

  1. display: grid;:

    أول خطوة لازم نعملها علشان نفعل شبكة grid هي نضيف الخاصية display: grid; للعنصر الأب، وده بيخلي كل العناصر اللي جواه تتحول تلقائيًا إلى grid items.

  2. grid-template-columns وgrid-template-rows:

    الخصائص دي بتحدد عدد الأعمدة والصفوف وحجمهم داخل الشبكة.

    • grid-template-columns: repeat(4, 25%); → 4 أعمدة كل واحد بعرض 25%.
    • grid-template-rows: repeat(3, 150px); → 3 صفوف ارتفاع كل صف 150px.

    ممكن كمان نستخدم fr لتقسيم المساحة بالتساوي زي: grid-template-columns: repeat(3, 1fr);.

  3. grid-gap:

    الخاصية grid-gap (أو gap في الإصدارات الحديثة) بتضيف مسافات بين الصفوف والأعمدة. مثال: grid-gap: 10px 25px; الرقم الأول للمسافة الرأسية، والتاني للأفقية.

  4. grid-column وgrid-row:

    الخصائص دي بتتحكم في مكان كل عنصر داخل الشبكة، وتقدر تخلي عنصر يمتد على أكتر من عمود أو صف.

    • grid-row: 1 / 4; → العنصر يبدأ من الصف الأول وينتهي عند الرابع.
    • grid-column: 1 / 3; → العنصر يمتد على أول عمودين.

    مثال من الكود: .div2 { grid-row: 1 / 4; grid-column: 1 / 3; }

  5. minmax() وauto:

    بنستخدم minmax(min, max) لما نحب نخلي حجم الصف أو العمود مرن بين حد أدنى وحد أقصى. مثال: grid-auto-rows: minmax(150px, auto); يعني أقل ارتفاع للصف 150px، لكن ممكن يزيد لو المحتوى أكبر.

  6. repeat():

    بدل ما نكتب نفس القيمة أكتر من مرة، بنستخدم repeat() لتكرارها بسهولة.
    مثال: grid-template-columns: repeat(4, 25%);

  7. grid-template-areas (مقدمة):

    اتكلمنا بشكل سريع عن grid-template-areas واللي بتسمح لنا نرسم التخطيط الكامل للشبكة بأسماء منطقية زي header وmain وfooter — هنشرحها بالتفصيل في المحاضرة الجاية.

Notes

  • 💡 استخدم fr بدل النسب المئوية علشان الشبكة تكون مرنة مع تغيير حجم الشاشة.
  • 🧩 تقدر تدمج grid مع flexbox لعمل تصميمات أكثر احترافية.
  • ⚙️ خاصية grid-auto-rows مفيدة لما العناصر أكتر من عدد الصفوف المحدد.
  • 🎨 CSS Grid بتديك تحكم كامل في تخطيط الصفحة بسهولة ووضوح.

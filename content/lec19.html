Lecture 19: CSS Animation & Keyframes

Summary

المحاضرة دي كانت عن واحدة من أقوى خصائص الـCSS وهي خاصية animation اللي بتخلينا نضيف حركات ديناميكية لأي عنصر في الصفحة. اتكلمنا عن كيفية تعريف الحركة باستخدام @keyframes، والتحكم في مدة وتكرار واتجاه وسرعة الحركة. كمان شرحنا الفرق بين أنواع animation-timing-function المختلفة زي ease، linear، ease-in، ease-out، وease-in-out.

Key Points

  1. @keyframes:

    الكلمة المفتاحية @keyframes بنستخدمها علشان نعرّف الحركة اللي العنصر هيعملها أثناء الأنيميشن. كل حركة بتتحدد بنسب مئوية من مدة التنفيذ (من 0% إلى 100%):

    @keyframes change {
    0% { background-color: yellow; }
    50% { background-color: blue; }
    100% { background-color: black; left: 1110px; } }

    ده مثال لحركة بتغير لون العنصر وتخليه يتحرك من اليسار لليمين تدريجيًا.

  2. animation-name & duration:

    الخاصيتين دول أساس أي حركة في CSS:

    • animation-name: اسم الحركة اللي عرفناها بـ@keyframes.
    • animation-duration: المدة اللي هتستغرقها الحركة (مثلاً 3s = ثلاث ثواني).

    مثال: animation-name: change; animation-duration: 3s;

  3. animation-timing-function:

    الخاصية دي بتتحكم في سرعة الحركة أثناء تنفيذها. شفنا الأنواع المختلفة منها:

    • linear → سرعة ثابتة من البداية للنهاية.
    • ease → بطيئة في البداية والنهاية وسريعة في النص (القيمة الافتراضية).
    • ease-in → بطيئة في البداية.
    • ease-out → بطيئة في النهاية.
    • ease-in-out → بطيئة في البداية والنهاية معًا.

  4. animation-iteration-count & delay:

    بنستخدم animation-iteration-count لتحديد عدد مرات تكرار الحركة: infinite بتخليها تتكرر للأبد. وكمان ممكن نضيف animation-delay لتأخير بدء الحركة.

    مثال: animation-delay: 4s; animation-iteration-count: infinite;

  5. animation-direction:

    الخاصية دي بتتحكم في اتجاه تكرار الحركة:

    • normal → الاتجاه الطبيعي (من البداية للنهاية).
    • reverse → بالعكس (من النهاية للبداية).
    • alternate → ذهاب وعودة بالتناوب.
    • alternate-reverse → تبدأ بالعكس ثم ترجع.

  6. animation-fill-mode:

    بتتحكم في حالة العنصر قبل وبعد الأنيميشن:

    • none → بيرجع للوضع الطبيعي بعد انتهاء الحركة.
    • forwards → بيحتفظ بآخر حالة بعد الانتهاء.
    • backwards → بياخد أول حالة قبل ما يبدأ.
    • both → بيجمع بين الحالتين.

  7. animation-play-state:

    الخاصية دي بتتحكم في تشغيل أو إيقاف الحركة:

    • running → الحالة الافتراضية (الأنيميشن شغال).
    • paused → توقف مؤقت.

    ممكن نستخدمها مع :hover مثلاً:

    .div1:hover { animation-play-state: paused; }

Notes

  • 💡 استخدم @keyframes لما تكون عايز تتحكم في أكتر من خاصية أثناء الحركة.
  • ⚙️ دمج خصائص الأنيميشن في سطر واحد بيكون كده: animation: name duration timing-function delay iteration-count direction fill-mode;
  • 🎨 استخدم ease-in-out للحركات الانسيابية لأنها بتدي مظهر احترافي.
  • 🧠 خاصية animation-play-state مفيدة لو حبيت توقف الأنيميشن مؤقتًا وقت التفاعل.

Lecture 27: Conditions, Logical Operators & Functions in JavaScript

Summary

في المحاضرة دي اتعلمنا إزاي نتحكم في سير البرنامج باستخدام الشروط (conditions) زي if وswitch، واتكلمنا كمان عن العمليات المنطقية (Logical Operators) اللي بتخلينا نربط أكتر من شرط مع بعض. وفي الجزء التاني من المحاضرة بدأنا نفهم يعني إيه functions وليه بنستخدمها لتقسيم الكود وتنظيمه.

Key Points

  1. If Condition:

    بنستخدمها علشان ننفّذ جزء معين من الكود لو الشرط اتحقق. الصيغة العامة:

    if (condition) { ... } else if (...) { ... } else { ... }

    مثال:

    var deg = prompt("enter the degree");
    if (deg >= 85) { console.log("Excellent"); }
    else if (deg >= 75) { console.log("Very Good"); }
    else if (deg >= 65) { console.log("Good"); }
    else if (deg >= 50) { console.log("Accepted"); }
    else { console.log("Failed"); }

  2. Logical Operators:

    العمليات المنطقية بتستخدم لدمج الشروط داخل if، وهي:

    • && → AND (كل الشروط لازم تكون صحيحة)
    • || → OR (يكفي شرط واحد يكون صحيح)
    • ! → NOT (يعكس النتيجة المنطقية)

    مثال:
    var x = 50, y = 100;
    if (x == 50 && y == 100) { console.log("True"); }
    else { console.log("False"); }

  3. Comparison Operators:

    • == → مقارنة القيم فقط
    • === → مقارنة القيم والنوع معًا
    • != → لا يساوي
    • > , < , >= , <= → للمقارنات الحسابية

  4. Switch Case:

    بنستخدم switch لما يكون عندنا قيم متعددة ممكن يحصل بناءً عليها تنفيذ مختلف.

    مثال:

    var x = prompt("enter your choice");
    switch(x) {
      case "a": console.log("Car"); break;
      case "b": console.log("iPhone 17"); break;
      case "c": console.log("10,000 LE"); break;
      default: console.log("Invalid choice");
    }

  5. Functions:

    Functions هي وسيلة لإعادة استخدام الكود بدل ما نكرره، وبتتكون من تعريف (definition) واستدعاء (call).

    الصيغة:

    function name(parameter1, parameter2) { ... }

    مثال:
    function multiply(a, b) { console.log(a * b); }
    multiply(8, 10);

    كمان ممكن نعمل دوال بتجمع أو تطرح أو تقسم بنفس الطريقة:
    function sum(x, y) { console.log(x + y); }
    function divide(a, b) { console.log(a / b); }

Notes

  • 🧠 استخدم if لما يكون عندك شرط واحد أو أكتر بشكل بسيط.
  • 💡 استخدم switch بدل if لما تكون القيم محددة مسبقًا.
  • ⚙️ الدوال بتخليك تعيد استخدام الكود وتخليه منظم أكتر.
  • 🚀 جرب تعمل دالة تجمع بين شرط ودالة – زي دالة بتحسب النتيجة وتطبع التقدير.

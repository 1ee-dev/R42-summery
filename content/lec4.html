Lecture 4: CSS Types, Selectors & Lists

Summary

المحاضرة دي كانت عن أنواع طرق كتابة تنسيقات CSS في صفحات الويب، واتكلمنا عن الفرق بين inline وinternal وexternal stylesheets، وإزاي نستخدم class وid لتطبيق تنسيقات معينة على عناصر محددة. كمان اتعلمنا إزاي نعمل القوائم (المرتبة وغير المرتبة) باستخدام <ul> و<ol>، وأخدنا فكرة عن خصائص زي border وborder-radius اللي بتدي شكل جميل للعناصر.

Key Points

  • CSS Types:
    فيه 3 طرق نكتب بيهم التنسيقات:
    • Inline: التنسيق بيتكتب جوه التاج نفسه باستخدام خاصية style.
    • Internal: بيتكتب داخل صفحة HTML نفسها جوه تاج <style>.
    • External: بنربط ملف .css خارجي بالصفحة عن طريق <link>، ودي أفضل طريقة عشان الكود يكون منظم وسهل التعديل.
  • Selectors:
    اتكلمنا عن class وid:
    • class بتستخدم لما عايز تطبق نفس التنسيق على كذا عنصر، وبتتكتب في CSS بنقطة (.) قبل الاسم.
    • id بتستخدم لو عايز تطبق تنسيق على عنصر واحد بس، وبتتكتب في CSS بعلامة (#) قبل الاسم.
  • Buttons Styling:
    اتكلمنا عن إزاي نغيّر شكل الأزرار باستخدام خصائص زي:
    • background-color لتحديد لون الخلفية.
    • color لتحديد لون النص.
    • border لتحديد نوع الإطار (زي solid أو dotted أو dashed).
    • border-radius لتدوير الحواف وتخلي شكل الزرار ناعم أكتر.
  • Lists:
    اتعلمنا نعمل نوعين من القوائم:
    • <ul> – Unordered List: قائمة بنقط.
    • <ol> – Ordered List: قائمة مرقّمة أو بحروف.
    وشرحنا كمان الخاصية type اللي بتغيّر شكل الترقيم في ol.
  • CSS Colors & Backgrounds:
    اتكلمنا عن استخدام background-color لتغيير خلفية العناصر، وكمان عن color لتغيير لون النصوص. في الأمثلة شفنا ألوان زي الأحمر، الأزرق، والأخضر بتتطبّق على العناوين والأزرار.

Notes

💡 حاول دايمًا تشتغل بـexternal CSS عشان التنسيق يكون منفصل وسهل التعديل.
⚙️ استخدم class لو هتكرّر نفس التنسيق، وid لو التنسيق لمرة واحدة.
🎨 جرب تغيّر في ألوان background وborder بنفسك عشان تشوف التأثير العملي.
🧠 كل محاضرة بتقربك من إنك تبني موقع منسّق واحترافي بالكامل.

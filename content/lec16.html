Lecture 16: Input Attributes, Required Fields & HTML Entities

Summary

المحاضرة دي ركزت على خصائص جديدة ومهمة في <input>، واللي بتساعد في التحكم في طريقة إدخال البيانات وتفاعل المستخدم معاها. كمان اتكلمنا عن عناصر إضافية زي <video> و<audio> وشفنا ازاي نقدر نستخدم أكثر من مصدر في نفس الوسائط. وأخيرًا، اتكلمنا عن HTML Entities اللي بنستخدمها لعرض رموز خاصة زي <, >, © من غير ما المتصفح يفسرها ككود.

Key Points

  • Input Attributes:
    اتكلمنا عن خصائص كتير بتأثر على سلوك الحقول:
    • required → بيجبر المستخدم يدخل قيمة قبل الإرسال.
    • readonly → بيخلي الحقل قابل للقراءة فقط، لكن قيمته تتبعت مع الفورم.
    • disabled → بيوقف الحقل تمامًا، ومش بيتبعت في الفورم.
    • multiple → بيسمح باختيار أكتر من ملف أو قيمة (زي في file).
    • size → بيحدد عرض الحقل بعدد الحروف الظاهرة.
    • value → القيمة الافتراضية داخل الحقل.
    مثال من الكود:
    <input type="text" value="Pay attention!!" readonly>
  • :required & :optional:
    دي pseudo-classes بنستخدمها علشان ننسق الحقول الإلزامية أو الاختيارية.
    • input:required { background-color: red; } → بيخلي الحقل الإجباري يظهر بلون مختلف.
    • input:optional { background-color: greenyellow; } → للحقل الاختياري.
  • Input Validation Attributes:
    زي minlength وmaxlength لتحديد أقل وأقصى عدد حروف في الحقل. مثال:
    <input type="password" minlength="8" maxlength="16">
    لو المستخدم كتب أقل من 8 حروف، المتصفح هيعرض رسالة خطأ تلقائيًا.
  • Self-Closing Tags:
    اتكلمنا عن العناصر اللي بتقفل نفسها ومش بتحتاج تاج إغلاق، زي:
    • <img> – لعرض الصور.
    • <source> – لتحديد مصدر الفيديو أو الصوت.
    • <br> – للنزول لسطر جديد.
    • <wbr> – بيسمح بكسر الكلمة الطويلة لو احتاج المتصفح.
    source بتتكتب داخل <video> أو <audio> كالتالي:
    <video width="380" controls>
      <source src="videos/mov.mp4" type="video/mp4" />
      <source src="videos/mov.ogg" type="video/ogg" />
    </video>
  • HTML Entities:
    لما نحب نعرض رموز خاصة في الصفحة زي < و > أو علامات © و®, بنستخدم Entities بالشكل ده:
    • &lt; → <
    • &gt; → >
    • &nbsp; → مسافة إضافية.
    • &copy; → © (حقوق الملكية).
    مثال:
    <h2>Islam &nbsp; &nbsp; Mohamed</h2>
    هتظهر في المتصفح كده:
    Islam     Mohamed

Notes

💡 دايمًا استخدم required مع الحقول المهمة في الفورم.
🎯 الفرق بين readonly وdisabled إن الأول بيتبعت في الفورم والتاني لا.
🎵 عناصر video وaudio بتحتاج مصادر بديلة علشان تشتغل في كل المتصفحات.
🧠 استخدم &entities; لما تحب تكتب رموز خاصة من غير ما المتصفح يعتبرها كود.

Lecture 26: Data Types & Operators in JavaScript

Summary

في المحاضرة دي بدأنا نتعمق في لغة JavaScript واتكلمنا عن أنواع البيانات المختلفة اللي ممكن نخزنها في المتغيرات، واتعرفنا على أهم العمليات الحسابية والمنطقية اللي بنستخدمها داخل الكود. الهدف كان إننا نفهم إزاي الـ JavaScript بتتعامل مع القيم وأنواعها.

Key Points

  1. أنواع البيانات (Data Types):

    أنواع البيانات في JavaScript بتحدد شكل القيمة اللي بنخزنها في المتغير، وأشهر الأنواع هي:

    • Number – الأرقام زي 10 أو 3.14
    • String – النصوص بين علامات تنصيص زي "Hello"
    • Boolean – القيم المنطقية (true / false)
    • Array – مجموعة بيانات مرتبة داخل أقواس مربعة [ ]
    • Object – بيانات غير مرتبة داخل أقواس معقوفة { }
    • Null – قيمة فارغة (لا شيء)
    • Undefined – متغير متعرف لكن لم يُسند له أي قيمة

  2. المصفوفات (Arrays):

    المصفوفة هي مجموعة من القيم المرتبة، كل عنصر له رقم فهرس (index) يبدأ من 0.

    var x = ["ahmed", "islam", 500, true, "eman"];

    x[0] → ترجع أول عنصر في المصفوفة، وx.length → ترجع عدد العناصر.

  3. الكائنات (Objects):

    الكائن بيخزن بيانات في شكل أزواج (اسم وقيمة)، وبيُستخدم لتنظيم المعلومات بشكل أوضح.

    var person = { name: "islam", age: 29, city: "mansoura" };

    الوصول لأي خاصية بيكون عن طريق النقطة: person.city

  4. القيم الخاصة Null و Undefined:

    • null معناها إن القيمة فاضية تمامًا.
    • undefined معناها إن المتغير اتعرف لكن ملحقش ياخد قيمة.

    مثال:
    var x = null; ← القيمة فاضية
    var y; ← القيمة undefined

  5. العمليات الحسابية (Arithmetic Operators):

    دي العمليات اللي بنستخدمها في الحساب داخل الكود:

    • + → الجمع
    • - → الطرح
    • * → الضرب
    • / → القسمة
    • % → باقي القسمة
    • ++ → زيادة واحد (increment)
    • -- → نقصان واحد (decrement)
    • ** → الأس (exponent)

    مثال:
    var num1 = 10;
    var num2 = 4;
    console.log(num1 ** num2); // 10000

  6. العمليات المختصرة (Assignment Operators):

    دي طريقة مختصرة للجمع أو الطرح والتخزين في نفس الوقت.
    زي:

    • x += 5 ← نفس معنى x = x + 5
    • x -= 3 ← نفس معنى x = x - 3
    • x *= 2 ← نفس معنى x = x * 2
    • x /= 4 ← نفس معنى x = x / 4

Notes

  • 🧠 استخدم typeof علشان تعرف نوع أي متغير.
  • ⚙️ Array بتتعامل بترتيب، أما Object فغير مرتب.
  • 💡 الفرق بين null و undefined إن الأولى فاضية عمدًا، والتانية لسه متحطش فيها قيمة.
  • 💪 جرب العمليات بنفسك في console علشان تتعود على النتايج.

Lecture 28: IIFE, Events & Loops in JavaScript

Summary

المحاضرة دي كانت مليانة مفاهيم مهمة في الجافاسكريبت زي الـIIFE أو الـSelf Invoked Function، وبدأنا نتعامل مع الأحداث (Events) زي الكليك وحركات الماوس والكيبورد. وفي الجزء الأخير اتكلمنا عن الحلقات التكرارية (Loops) اللي بنستخدمها علشان نكرر كود معين أكتر من مرة بشكل منظم.

Key Points

  1. IIFE (Self Invoked Function):

    اختصار لـ Immediately Invoked Function Expression، وهي دالة بتتكتب وبتتنفّذ في نفس اللحظة، ومش بتحتاج استدعاء.

    بتستخدمها علشان تعزل الكود (Encapsulation) وتمنع تداخل المتغيرات بين الملفات.

    مثال:

    (function code1() {
      console.log("islam");
    })();

  2. Events (الأحداث):

    الأحداث بتخليك تتفاعل مع المستخدم — زي لما يضغط على زر، أو يمر الماوس على عنصر، أو يكتب حاجة في حقل إدخال.

    • click → عند الضغط مرة واحدة
    • dblclick → عند الضغط مرتين
    • mouseenter وmouseleave → دخول وخروج الماوس
    • mousemove → حركة الماوس
    • keydown وkeyup → عند الضغط على لوحة المفاتيح

    أمثلة:

    var button = document.getElementById("btn1");
    var div = document.getElementById("div1");

    function code1() { div.innerHTML = "islam"; }

    button.onclick = code1;
    // أو
    button.addEventListener("click", code1);

    مثال آخر باستخدام الماوس:

    var x = 0;
    button.onmousemove = function() {
      div.innerHTML = x++;
    };

  3. Loops (الحلقات التكرارية):

    بنستخدم الحلقات علشان ننفّذ نفس الكود أكتر من مرة، بدل ما نكرره يدويًا. عندنا 3 أنواع رئيسية:

    • for loop: لما نعرف عدد التكرارات مسبقًا.
      for (var x = 40; x <= 80; x++) { console.log(x); }
    • while loop: بتستمر طول ما الشرط صحيح.
      var x = 40;
      while (x <= 80) {
        console.log(x);
        x++;
      }
    • do...while loop: بتتنفّذ مرة واحدة على الأقل قبل ما تتحقق من الشرط.
      var x = 40;
      do {
        console.log(x);
        x++;
      } while (x <= 80);

    ⚠️ لو نسيت تحديث الشرط (زي x++) ممكن تقع في infinite loop (حلقة لا نهائية).

Notes

  • 💡 استخدم IIFE لما تكون عايز تعزل الكود عن باقي الصفحة.
  • 🧠 الأحداث هي أساس التفاعل مع المستخدم في صفحات الويب.
  • ⚙️ جرب تدمج الأحداث مع الحلقات — زي مثلاً زرار بيزود العداد كل مرة المستخدم يضغط عليه.
  • 🚫 دايمًا تأكد إن الحلقات فيها شرط خروج علشان متدخلش في infinite loop.
